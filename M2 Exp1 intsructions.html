Module 2

EXPERIMENT No 1:Set Theory


INSTRUCTIONS:


1) Interface Overview:
a) The main window provides input fields for two sets, buttons for calculating the union and intersection, and a label to display the result.


2) Entering Sets:
a) Enter the elements of Set 1 in the first text field. Separate the elements with commas (e.g., 1, 2, 3).
b) Enter the elements of Set 2 in the second text field using the same comma-separated format.


3) Calculating Union:
a) After entering the sets, click the "Calculate Union" button.
b) The result label will be updated to display the union of Set 1 and Set 2.


4) Calculating Intersection:
a) Alternatively, you can click the "Calculate Intersection" button to find the intersection of Set 1 and Set 2.
b) The result label will be updated to display the intersection.


5) Viewing Results:
a) The result label below the buttons will show the outcome of the set operation you performed.


6) Example:
a) For example, if Set 1 contains elements {1, 2, 3} and Set 2 contains elements {2, 3, 4}, the union will be {1, 2, 3, 4} and the intersection will be {2, 3}.


7) Closing the Application:
a) Close the application window when you are done using it.


Notes:
a) The simulator assumes that the sets are entered as comma-separated values without spaces.
b) The sets are case-sensitive, so "A" and "a" would be treated as different elements.
c) The simulator uses a graphical interface to make set calculations more user-friendly.
