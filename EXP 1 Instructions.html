EXPERIMENT No 1:Propositional Logic & Predicate Logic


INSTRUCTIONS:

1. Input Logic Statement:
- Enter your logic statement in the "Enter Logic Statement " text field.

2. Check Logic Type:
- Click the "Check Logic" button to analyze the entered logic statement.

3. View Result:
- The result of the logic check will be displayed in the scrollable text area below the input field.

4. Interpretation of Results:
- If the statement contains logical symbols such as ∧, ∨, ¬, →, or ↔, it will be identified as "Propositional Logic.
- If the statement includes parentheses and quantifiers like ∀ (for all) or ∃ (there exists), it will be identified as "Predicate Logic.
- If the statement doesn't match either criteria, it will be labeled as "Not Recognized as Propositional or Predicate Logic.

5. Repeat:
- You can enter new logic statements and click "Check Logic" to analyze them.



Note:
1. Copy and paste these symbols for logical operators: ∧ (AND), ∨ (OR), ¬ (NOT), → (IMPLIES), ↔ (BICONDITIONAL), ∀ (universal quantifier), ∃ (existential quantifier).
2. The simulator is designed for simple logic checks based on the presence of specific symbols. It may only cover some aspects of formal logic
